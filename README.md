# NekoTune
A Spotify alternative... just for cats? No, wait... it's better than Spotify. This is NekoTune: the next generation of music streaming.

## Caratteristiche
- **Ricerca e Streaming Completo**: Usa l'API di YouTube Music in background.
- **Supporto Mobile e Desktop**: UI responsiva con player stile Spotify.
- **Avanzate Impostazioni Audio**: Equalizzatore, Qualità Audio, e Cache Manager Integrato.
- **Account e Preferiti**: Collegati con OAuth e importa i tuoi brani.
- **Interazione Artistica**: Follow, biografie automatiche e raccomandazioni intelligenti.

Built with React/Vite/Electron.
